各授業回の学習事項

第1回 WWW と HTML

特にHTML要素について学ぶことができました。内容としては、pやstrongなどで内容を間で挟むようにして使うことを知りました。 また、インライン要素（strongなど）の中にブロック要素（pなど）を入れてはいけないことも知りました。

演習の成果： 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

2回目の講義ではページを装飾する記述であるCSSについて学びました。色を変えたり枠線をつけたりと様々な装飾ができることを知りました。 また、CSS の書式はセレクタ・プロパティ名・プロパティ値 で成り立っていることも知りました。

演習の成果： 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

「JavaScript」について詳しく知ることができました。1年生の頃に学んだJavaよりもJavaScriptの方が使いやすいと感じました。 コンソールで試してからプログラミングに付け足すことで、間違いを防ぐことができるということを学びました。

演習の成果： 演習3-6: オブジェクトの参照
